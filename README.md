# GLOWPUTE

> Turn Idle GPUs into Verifiable AI Compute.

Glowpute is a decentralized GPU compute network built on Solana that transforms idle consumer GPUs into verifiable AI inference infrastructure.

Install the Glowpute Node, contribute unused GPU resources, execute TEE-attested inference workloads, and earn rewards automatically through decentralized compute markets.

# 🚀 What is Glowpute?

Modern AI depends on massive GPU infrastructure controlled by a handful of cloud providers.

Meanwhile, millions of consumer GPUs remain idle for most of the day.

Glowpute connects these unused GPUs into a decentralized compute network where contributors earn rewards by serving AI inference workloads with cryptographically verifiable execution.

Every completed inference is measured, verified, and settled on Solana.
